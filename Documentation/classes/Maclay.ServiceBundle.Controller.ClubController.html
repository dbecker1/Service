manageClubAction()

manageClubAction(integer  $quarter, boolean  $export = false)

The method for the club sponsor's home page.

This is the main page of the club sponsor. It shows each memeber and their total number of approved. There is then a dropdown to narrow down the hours by quarter and then an export function that will export them to a CSV file that is readable by Excel.

Parameters

integer $quarter

Optional. Used to narrow down approved totals.
boolean $export

Optional. Used to export approved totals.

addUserToClubAction(integer  $userId, integer  $clubId)

batchEnterRecordAction(\Maclay\ServiceBundle\Controller\Reqest  $request)

The method for entering records for members in batch.

This method is for when club sponsors want to enter multiple records with the same details for multiple users. The sponsors have the freedom to put in a number of hours for each individual student.

getStudentsByLastNameAction()

getStudentsByLastNameAction(string  $lastName) : array

The method to get users by their last name.

This method is called by Ajax from the batchEnterRecord page. When a club sponsor types in a last name in to a text box, this method returns a list of students with that last name. If there are multiple students, then there is a dropdown list. If there is only one, then it simply shows the first name.

Parameters

string $lastName

The last name of the desired students

Returns

array —

$users The users matching the last name.
